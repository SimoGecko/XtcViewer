# XTC Viewer

A simple web-based viewer for **XTC/XTCH/XTG/XTH files**, which contain grayscale images (1-bit or 2-bit e-paper format). View pages, zoom in/out, and navigate chapters.

---

## Features

- Load `.xtc/.xtch/.xtg/.xth` files directly in the browser  
- Supports 1-bit (monochrome) and 2-bit (grayscale) images  
- Zoom in/out with percentage input  
- Navigate pages with previous/next buttons  
- Scrollable chapter list with clickable chapters  

---

## Usage

1. Open `index.html` in your browser  
2. Click **Choose File** and select a file  
3. Use the arrows or chapter list to navigate pages  
4. Adjust zoom using the input (%)  

---

## Notes

- Works entirely in the browser (no server required)  
- Compatible with modern browsers  
- Based on the file specification from [here](https://gist.github.com/CrazyCoder/b125f26d6987c0620058249f59f1327d)
- Convert your EPUBs to XTC/XTH [here](https://x4converter.rho.sh/)
